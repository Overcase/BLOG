```
vim 使用
vi/vim [fileName]
i 进入编辑
Esc退出编辑模式，输入以下命令：
:wq保存后退出vi，若为 :wq! 则为强制储存后退出（常用）
:w保存但不退出（常用）
:w!若文件属性为『只读』时，强制写入该档案
:q离开 vi （常用）
:q!若曾修改过档案，又不想储存，使用 ! 为强制离开不储存档案。
:e!将档案还原到最原始的状态！

```
